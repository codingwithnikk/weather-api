# Weather App By Using API

This is a web app using HTML, CSS, JS fetching data from web api. API name called [Open Weather API](https://openweathermap.org/api).

## Tech Used
- HTML
- CSS
- Javascript
- Open Weather API

## JS Methods Used
- fetch method
- Event Listener
- Query Selectors
